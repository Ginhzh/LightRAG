# 普通 RAG vs Graph-RAG 子代理评分表

## 评估说明

- 本表不使用 DeepSeek 或外部评审 API，由 5 个 Codex 子代理按题号区间并行评分后合并。

- 每个子代理只依据两份结果文件中的 question、category、expected_keywords、expected_answer_points、answer、sources 判断。

- 评分标准：0=错误或没有回答；1=部分正确但遗漏严重；2=基本正确但解释不完整；3=正确、完整、有依据；4=正确、完整、有关系链、有明确排查建议或可执行判断路径。

- “是否包含关系链”按子代理原始描述保留，通常同时说明普通 RAG 与 Graph-RAG 的链路强弱。

- 该评分比 DeepSeek 自动评审更严格，尤其会惩罚错引、检索失效、泛化回答和偏题扩展。
